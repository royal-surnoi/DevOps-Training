### Troubleshoot 
- CPU memory 

    ```top```

- Hard Disk 
    
    ```df -h```

- RAM status 

   ```free -mh```

- Process running Status

    ```ps -ef | grep <process_name> ```

- Ports enable status

    ``` netstat -lntp```

- Service status 

    ```systemctl status <service>```

- Firewall status (security groups) check

- Check Internet
    ```ping <URL/IP Address> ```
